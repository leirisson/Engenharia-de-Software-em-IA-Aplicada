# Feature Overview
Gerencia sessões de chat: criar, encerrar e recuperar estado/histórico por sessionId.

# Actors
- Usuário
- Serviço de chat

# Main Use Cases
- Criar sessão
- Encerrar sessão
- Obter status e metadados da sessão

# API Endpoints (if applicable)
- POST /chat/session
- DELETE /chat/session/:sessionId
- GET /chat/session/:sessionId
